# 资源文件目录

本目录用于存放项目相关的静态资源：

## 📸 建议添加的内容

### 界面截图
- `main_interface.png` - 主界面截图
- `settings_window.png` - 设置窗口截图
- `duplicate_groups.png` - 重复视频组展示

### 演示GIF
- `quick_start.gif` - 快速开始操作演示
- `comparison_modes.gif` - 三种比对模式对比演示

### 架构图
- `architecture.png` - 系统架构图
- `cache_system.png` - 缓存系统工作流程图

## 📝 截图建议

1. **分辨率**: 使用 1920x1080 或更高
2. **格式**: PNG（清晰）或 GIF（动态演示）
3. **内容**: 
   - 显示完整的窗口界面
   - 包含典型的使用场景
   - 突出核心功能区域
